## アプリケーション詳細
ファッション関連のセレクトショップやブランドを探すことのできるアプリケーションになります。<br>
```Vue.js```と```Ruby on Rails```のAPIモードを使用したSPA(シングルページアプリケーション)です。<br>
仮想環境構築には```Docker```を使用し、テスト環境には```Circle Ci```を導入しました。<br>
```vue-router```プラグインを導入し、ルーティング制御を行っています。

・Circle Ci 2.1<br>
